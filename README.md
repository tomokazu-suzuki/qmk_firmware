## OK60 HHKB

* default
    - Caps lock key assigned to MO(1) + Tab key

    `make ok60_hhkb:default`

* idle_timer
    - Turn the backlight off automatically in 30 seconds.
    - Turn LEDs off automatically in 5 minutes.

    `make ok60_hhkb:idle_timer`
